## Getting Started

### Prerequisites

- Unity Hub
- Unity 2021.3 LTS
- Git with Git LFS
- Visual Studio or VS Code

### Setup

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd idle
   ```

2. Install Git LFS:
   ```bash
   git lfs install
   ```

3. Open Unity Hub and add the `InfinityIdle` project

4. Install required packages:
   - TextMeshPro
   - BreakInfinity.cs
   - Newtonsoft.Json
